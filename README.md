**CondoHub** es una plataforma **SaaS moderna** diseñada para revolucionar la administración de condominios. Combina **tecnología avanzada, diseño premium y facilidad de uso** para ofrecer una experiencia eficiente tanto a administradores como a residentes.

---

## ✨ ¿Por qué CondoHub?

CondoHub no es solo un sistema de gestión — es una solución integral que optimiza procesos, mejora la comunicación y eleva la calidad de vida dentro de cualquier comunidad residencial.

---

## 🚀 Características Destacadas

### 🎨 Interfaz de Nueva Generación
- **Glassmorphism UI**: Diseño elegante con efectos de cristal y desenfoque  
- **Animaciones dinámicas**: Implementadas con AOS (Animate On Scroll)  
- **100% Responsive**: Adaptado a móviles, tablets y escritorio  

---

### 🧩 Módulos Inteligentes

#### 🔐 Control de Accesos
- Registro detallado de entradas y salidas  
- Compatible con sistemas biométricos  

#### 💰 Gestión Financiera
- Control transparente de pagos  
- Estados de cuenta en tiempo real  

#### 🏊 Reserva de Amenidades
- Reservas online de áreas comunes  
- Calendario inteligente de disponibilidad  

#### 📢 Comunicación Activa
- Notificaciones masivas  
- Mensajería directa entre administración y residentes

## ⚙️ Instalación

Sigue estos pasos para ejecutar el proyecto en tu entorno local:

```bash

git clone https://github.com/tu-usuario/Pagina-Principal-Condominio.git

# Entrar al proyecto
cd Pagina-Principal-Condominio

# Instalar dependencias
npm install

# Ejecutar en desarrollo
npm run dev
