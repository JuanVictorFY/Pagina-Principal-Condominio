# 🏢 CondoHub - Gestión Residencial de Élite

CondoHub es una plataforma SaaS de vanguardia diseñada para transformar la administración de condominios. Enfocada en la simplicidad, la seguridad y el diseño de alto nivel, ofrece una experiencia integral tanto para administradores como para residentes.

## ✨ Características Principales

### 🚀 Interfaz de Vanguardia
- **Glassmorphism Design:** Tarjetas con efectos de cristal esmerilado y desenfoque de fondo.
- **Experiencia Inmersiva:** Animaciones fluidas utilizando la librería AOS (Animate On Scroll).
- **Responsive Design:** Totalmente optimizado para dispositivos móviles, tablets y escritorio.

### 🛠️ Módulos del Sistema
- **Control de Accesos:** Registro detallado de ingresos y salidas con tecnología biométrica.
- **Gestión Financiera:** Transparencia total en cuotas y estados de cuenta en la nube.
- **Reserva de Amenidades:** Sistema inteligente para la gestión de áreas comunes.
- **Comunicación Activa:** Mensajería masiva y notificaciones push para la comunidad.

---

## 🛠️ Tecnologías Utilizadas

- **Frontend:** React.js con Vite para un desarrollo ultra rápido.
- **Estilos:** Bootstrap 5 personalizado con CSS3 avanzado (Variables, Grids, Flexbox).
- **Animaciones:** AOS.js y transiciones CSS personalizadas.
- **Componentes:** React-Slick para carruseles de alta fidelidad.
- **Iconografía:** Bootstrap Icons.
